# Data-Mining-Fudamentals #

Project for the Data Mining 1 course. It consists of analyzing and processing tabular data, employing fudamental data mining and machine learning algorithms on the Sportify dataset.

## Focus Areas
- **Data Understanding and Preparation:** Data Semantics, Missing value, Visualization the distribution of data, Assessing data quality, Pairwise correlations and eventual elimination of variables

- **Clustering:** K-means, DBSCAN, Hierachical Clustering

- **Classification:** KNN, Naive Bayes, Decision Tree

- **Pattern Mining:** Frequent Pattern, Association Rules

- **Regression:** Univariate and multivariate regression with Linear, Ridge, Lasso
